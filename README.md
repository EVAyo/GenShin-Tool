# Genshin Tools

## Getting Started

**Requirements**

* `node@16+` (esmodule needed) and `pnpm`
* `flutter@2.10+`

```shell
# install dep
make install
# process genshin data
make convert.genshin-data

# build android
make build.android

# build ios
make build.ios
```

### Dev

```shell
# for watching & generate
make gen
```

use IDE to run in devices.
