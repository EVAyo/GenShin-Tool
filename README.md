# GenshinArtScanner

[ProblemFactory/GenshinArtScanner](https://github.com/ProblemFactory/GenshinArtScanner)
的项目的一些继续开发

- 增加 GUI
- 重构代码
- 增加功能


## 用法

1. 双击 ArtScanner.exe 打开程序，等待一会儿代码的解压。
2. 将原神调整分辨率为 1600 * 900，打开背包 - 圣遗物。
3. 捕获窗口
4. 调整扫描选项
5. 开始扫描
