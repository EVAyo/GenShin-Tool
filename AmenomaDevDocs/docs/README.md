原名GenshinArtScanner，用于扫描原神背包页面的圣遗物并导出json信息，供圣遗物分析工具分析。目前支持[莫娜占卜铺](https://www.mona-uranai.com)、[MingyuLab](https://genshin.mingyulab.com/)、[Genshin Optimizer(仅限英文)](https://frzyc.github.io/genshin-optimizer/#/)。
